Adding New Slide Decks

🛠️ 新しいスライドを追加する方法

新しいプレゼンテーションを作成するには、packages/slides/slides/ ディレクトリの下に新しいフォルダーを作成し、スライドを配置します。

# 1. スライド用ディレクトリを作成する (例: my-slide)
mkdir -p packages/slides/slides/my-slide

# 2. その中に index.html を作成してスライドを記述する
touch packages/slides/slides/my-slide/index.html

# 3. 本インデックス(packages/slides/index.html)にカードを追記する
