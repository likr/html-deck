# create-html-deck

Boilerplate initializer for HTML-Deck presentations.

## Usage

You can scaffold a new HTML-Deck presentation project using the following command:

```bash
npm create html-deck <project-directory>
```

Alternatively, you can run:

```bash
npx create-html-deck <project-directory>
```

## Options

- `--aspect <ratio>`: Set the slide aspect ratio. Examples: `16:9` (default), `4:3`, `1.6`.
- `--lang <lang>`: Set the HTML `lang` attribute. Examples: `ja`, `en` (default).
- `--presenter`: Scaffold a local presenter view dashboard (`presenter.html`) and link it.

## Getting Started in the Scaffolded Project

After the scaffolding process is complete, navigate into the project directory, install the required dependencies, and start the local development server:

```bash
cd <project-directory>
npm install
npm run dev
```
