🤖

Agent-First Design

Flat markup avoids the deeply nested section tags typical of Reveal.js. Simple layout attributes like layout="split" prevent AI syntax errors.

⚡

Build-Free

No bundling, Vite, or npm scripts needed. Just import html-deck.js and link html-deck.css into a plain HTML file, and you are ready to write slides.

📶

Offline Support

Math equations (KaTeX) and syntax highlighting (PrismJS) are resolved from local vendor files, enabling complete offline editing.

👤

Presenter Synchronizer

Press 'P' to open a separate presenter dashboard. Uses BroadcastChannel to sync slides, elapsed clocks, and presenter notes.

🖨️

Flawless Printing

Prints cleanly out of the box. Automatically enforces aspect ratio constraints on A4 Landscape sheets and removes UI elements.

🎨

CSS Variable Customization

No opinionated theme constraints. Define colors, font sizes, transitions, and line-heights globally using custom properties.

Quick Start

Writing a presentation is as simple as creating an HTML file.

<!-- index.html -->
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>My Presentation</title>
  
  <!-- 1. Load HTML-Deck Script & Styles -->
  <script type="module" src="./src/html-deck.js"></script>
  <link rel="stylesheet" href="./src/html-deck.css">
  
  <!-- 2. Customize variables globally -->
  <style>
    :root {
      --hd-slide-bg: #0f172a;
      --hd-text-color: #f8fafc;
      --hd-primary: #3b82f6;
    }
  </style>
</head>
<body>

  <!-- 3. Declare slides -->
  <hd-deck transition="fade" presenter-url="./demo/tutorial/presenter.html" aspect-ratio="16:9">
    
    <!-- Title Slide -->
    <hd-slide layout="title">
      <hd-heading>Welcome to HTML-Deck</hd-heading>
      <hd-text>An elegant presentation engine</hd-text>
    </hd-slide>

    <!-- Bullet Slide with Speaker Notes -->
    <hd-slide layout="title-body">
      <hd-heading slot="title" level="2">Presentation Agenda</hd-heading>
      <hd-list>
        <li>WebComponents Introduction</li>
        <li>Declarative layouts</li>
        <li>KaTeX & PrismJS rendering</li>
      </hd-list>
      <hd-notes>
        <h2>Presenter notes</h2>
        <p>Emphasize that custom elements prevent library pollution.</p>
      </hd-notes>
    </hd-slide>

  </hd-deck>
</body>
</html>
