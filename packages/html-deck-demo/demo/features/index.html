Individual Contrast Override

Applying class="hd-primary hd-background" directly to <hd-slide> overrides its background and text colors completely.

This specific slide will remain styled with the contrast colors regardless of what global theme is loaded, making it perfect for section divider slides or key highlights.

Presenter Synchronization

This library uses a local BroadcastChannel synchronization protocol.

When you press P, a separate speaker dashboard opens in a popup window. It displays: elapsed stopwatch timer, current local system clock, slide index status, current slide previews, next slide previews, and custom notes.

The text you see in the presenter notes is fetched from the <div slot="notes"> element contained inside the slide HTML markup.

Slide 17: Presenter Notes
