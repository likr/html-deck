Individual Override: bg="primary"

This slide uses the bg="primary" attribute. Even when switching themes, this specific slide will use the theme's primary accent color as the background, and automatically swap text to high-contrast white.

This is perfect for divider slides, intro pages, or key takeaways that need to stand out from the rest of the deck.

  • Background: Primary accent color.
  • Text and headings: Pure white.
  • Math & Lists: Pure white.

Individual Override: bg="secondary"

Similarly, using bg="secondary" switches the slide background to the secondary theme color, maintaining clear legibility with white text.

Individual Override: invert

This slide uses the invert attribute. It dynamically swaps the background and body text colors according to the active theme.

For instance, in the default light theme, this slide becomes dark. In dark themes, it will become light.
