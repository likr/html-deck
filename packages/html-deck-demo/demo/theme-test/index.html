Individual Override: class="hd-primary hd-background"

This slide uses the class="hd-primary hd-background" classes. Even when switching themes, this specific slide will use the theme's primary background color, and automatically adapt text to the primary color.

This is perfect for divider slides, intro pages, or key takeaways that need to stand out from the rest of the deck.

  • Background: Primary background color.
  • Text and headings: Primary text color.
  • Math & Lists: Primary text color.

Individual Override: class="hd-secondary hd-background"

Similarly, using class="hd-secondary hd-background" switches the slide background to the secondary theme background color, maintaining clear legibility with the secondary color.
